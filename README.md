Sorter App lib
===============
This is a libery dor sorting array 

Usage
-----
SORT - sorting array by Ascending
RSIRT - sorting by Descending
Null - null object

Example:

$data = new StrategyContext($data);
$factory = new StrategyFactory();
$data->setStrategy($factory->construct('SORT'));
